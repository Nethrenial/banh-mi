BanhMi ToDo

BanhMi is a simple & experimental framework written for Bun runtime.It's currently only intended for personal use. It's not heavily tested and not recommended for production use and the api can change at anytime. This todo application is written with html, css, tailwindcss,javascript, banh-mi and mongoose (mongodb) to serve as an feature showcase of banh-mi framework.
